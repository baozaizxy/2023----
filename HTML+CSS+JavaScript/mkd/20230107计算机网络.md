mac地址可以理解为每个人的学号，在入校时已经决定（mac地址出厂时每个主机就有）

ip地址类似于班号（子网）+序号（主机号）

一个班级（子网）内的同学（主机）通过ARP协议广播比对ip获取每个同学的学号（mac地址）

询问mac的数据包，mac地址栏填写的时特使mac值，因此mac栏失效，通过ip找到目标

传输数据子网中主机获取数据包后，对比自身ip与数据包的目的地址ip，相同则接收数据
